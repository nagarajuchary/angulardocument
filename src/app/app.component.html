What is angular

    Angular is a development platform, built on TypeScript.
    
        As a platform, Angular includes:
        
        A component-based framework for building scalable web applications
        A collection of well-integrated libraries that cover a wide variety of features, 
        including routing, forms management, client-server communication, and more
        A suite of developer tools to help you develop, build, test, and update your code
        With Angular, you're taking advantage of a platform that can scale from single-developer 
        projects to enterprise-level applications. Best of all, the Angular ecosystem consists of a diverse group of over 1.7 million developers, library authors, and content creators.

#How set up Angular

  • Download vscode
  • Download node js
  • Install ANGULAR Cli

#How install node

  • Download Node js
  • Install node
  • Open command prompt in window
      • How to check node version
    • Open command prompt Enter command as(node -v)
    • click enter

#How to install ANGULAR CLI

  • After Node is installed
  • Open command prompt
  • Enter command as npm install -g @Angular/cli
  • ANGULAR is installed.
#How to create Project in Angular
  • Open window command prompt Enter command as ng new projectname
  • Eg: ng new angular (angular is the name of the project)
  • Now project from File Manager using command cmd
#How to run Project in angular
  • Open Terminal in VSCODE Enter command as ng serve
  • Click Enter
  • Project starts to Build
  • Now after building te project copy the URL available in terminal
  • Enter the url in chrome and press Enter
  • The project is displayed in the chrome

#How to install boostrap

  • Open terminal in the project
  • Enter command npm install boostrap to install bootstrap
  • Bootstrap is installed
  • Add bootstrap links in Angular.json file
  • Links "node_modules/bootstrap/scss/bootstrap.scss",
    "node_modules/bootstrap-icons/font/bootstrap-icons.css",
    "src/styles.scss"

How to create component

  • Open terminal in vscode
  • Enter the command ng generate component name
  • Eg:ng generate component header(header is the component name)
  • Component is Created.
  • Now add the TAG with component name in app.component.html
  • Eg:<app-component name> </app-component name>
  • Now add the component in Imports available in app.component.ts
  • Add router-outlet tag in app,component.html

How add routerLink

  • Open app.component.html add anchor tag using the attribute routerLink
  • Give the path in RouterLink
  • Open app.compoment and add routerLink in Imports
  • Now Open the app.routes.ts and declare the path in Routes
  • Eg:<a routerLink="pathname"></a>
  • Path must be in the curlybraces within the index
  • Eg:{path:"pathname",component:componentname}
